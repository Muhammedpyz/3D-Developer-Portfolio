# 3D Developer Portfolio

Bu proje, React, Three.js ve Tailwind CSS kullanılarak geliştirilmiş, etkileşimli ve 3D öğeler içeren modern bir geliştirici portfolyosudur.

## 🚀 Özellikler

*   **3D Etkileşim:** Three.js ile oluşturulmuş 3D modeller ve sahneler.
*   **Responsive Tasarım:** Mobil ve masaüstü cihazlarla tam uyumlu.
*   **Modern UI:** Glassmorphism, glitch efektleri ve özel animasyonlar.
*   **İletişim Formu:** EmailJS entegrasyonu ile çalışan fonksiyonel iletişim formu.

## 🛠️ Teknolojiler

*   [React](https://reactjs.org/)
*   [Three.js](https://threejs.org/)
*   [Tailwind CSS](https://tailwindcss.com/)
*   [Framer Motion](https://www.framer.com/motion/)
*   [Vite](https://vitejs.dev/)

## 📦 Kurulum ve Çalıştırma

Projeyi yerel ortamınızda çalıştırmak için aşağıdaki adımları izleyin:

1.  **Projeyi klonlayın veya indirin.**
2.  **Bağımlılıkları yükleyin:**
    ```bash
    npm install
    ```
3.  **Geliştirme sunucusunu başlatın:**
    ```bash
    npm run dev
    ```
4.  Tarayıcınızda `http://localhost:5173` adresine gidin.
